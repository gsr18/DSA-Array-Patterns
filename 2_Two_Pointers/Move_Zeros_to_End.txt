--> Key Idea: Shift non-zero elements left in one pass, preserving order, then fill the rest with zeros.

Why Two Pointers?
One tracks the next position (i), the other scans for useful values (j).

--> Approach:

If arr[j] != 0, swap with arr[i], increment i

--> Pseudocode

i = 0
for j = 0 to n-1:
    if arr[j] != 0:
        swap(arr[i], arr[j])
        i++
