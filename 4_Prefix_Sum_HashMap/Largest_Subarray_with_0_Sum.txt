--> Key Idea: If same prefix sum appears again â†’ subarray in between sums to 0

Why Prefix + HashMap?

--> Approach:

--> Visualisation

Prefix = 0 at index 1 and 5 â†’ subarray [2..5] = sum 0


--> Pseudocode

map = {}
prefix = 0, maxLen = 0
for i = 0 to n-1:
  prefix += arr[i]
  if prefix == 0:
    maxLen = i + 1
  if prefix in map:
    maxLen = max(maxLen, i - map[prefix])
  else:
    map[prefix] = i
