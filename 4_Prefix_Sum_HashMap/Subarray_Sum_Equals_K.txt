--> Key Idea: Count total subarrays with sum K using prefix sum frequency

Why Prefix + HashMap?
Counting how often prefix - K appeared before

--> Approach:
Count frequency of each prefix sum

--> Visualisation

--> Pseudocode

map = {0: 1}
prefix = 0, count = 0
for num in arr:
  prefix += num
  if prefix - K in map:
    count += map[prefix - K]
  map[prefix] += 1
